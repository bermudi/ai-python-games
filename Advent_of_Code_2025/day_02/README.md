# Advent of Code 2025 - Day 02 Solutions

This directory contains solutions for **Advent of Code 2025 Day 02** from various AI models. The problem involves finding "invalid product IDs" - numbers that consist of a digit sequence repeated either twice (Part 1) or at least twice (Part 2).

## Problem Summary

**Part 1**: Find the sum of all IDs that consist of a digit sequence repeated exactly twice (e.g., 1212, 3434).

**Part 2**: Find the sum of all IDs that consist of a digit sequence repeated at least twice (e.g., 121212, 123123123).

**Correct Answers**:
- Part 1: **44854383294**
- Part 2: **55647141923**
